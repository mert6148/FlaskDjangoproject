API Sunucu Yönetimi

1. Internet / Wi-Fi Durumu

GET /api/v1/wifi-check

2. Veritabanı Yapılandırması

POST /api/v1/db-config JSON yükü:

{ "db_url": "mysql+mysqlconnector://user:pass@host:3306/dbname" }

3. JSON ID Bağlama Protokolü

Aşağıdaki uç nokta item üretir ve id atar:

POST /api/v1/item

{ "name": "örnek" }
