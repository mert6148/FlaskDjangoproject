# src/docs Dökümantasyon Kılavuzu

Bu klasör `src` içindeki dil bazlı ve ortam konfigürasyonu dokümantasyonlarını toplar.

## Mevcut Belgeler

- `python.md` - Python uygulama ve yapılandırma rehberi
- `csharp.md` - C# proje ve Web API rehberi
- `java.md` - Java uygulama örnekleri
- `javascript.md` - JavaScript entegrasyon rehberi
- `html.md` - HTML ön yüz kullanımı
- `cpp.md` - C/C++ yapılandırma şablonu ve rehberi
- `.env.example` - Ortam değişkenleri şablonu

## Kullanım

Her dil için ilgili dosyayı açarak proje içinde hangi dosyaların nereye taşındığını ve nasıl çalıştırıldığını görebilirsiniz.

## Nasıl Güncellenir

- Yeni bir dil ekliyorsanız, `docs/` altına yeni bir `.md` dosyası ekleyin.
- Küçük yapılandırma notları için `CONFIG_README.md` ve `src/README.md` dosyalarını güncelleyin.
