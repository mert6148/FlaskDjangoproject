# HTML Dökümantasyonu

`src/html/` klasörü statik HTML sayfalarını barındırır.

## İçerik

- `index.html` - temel HTML ön yüz sayfası

## Kullanım

Bu dosya doğrudan tarayıcıda açılabilir veya bir web sunucusuna yerleştirilebilir.

### Basit kullanım

```bash
start src/html/index.html
```

veya Python sunucusu ile:

```bash
cd src/html
python -m http.server 8000
```

## Notlar

- HTML kaynakları `src/html/` altında toplanmıştır.
- Statik dosyalar için `src/static/` klasörü kullanılabilir.
