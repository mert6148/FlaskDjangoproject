# Java Dökümantasyonu

`src/java/` klasörü Java uygulama örneklerini içerir.

## İçerik

- `Application.java` - Java uygulama örneği
- `BankApplication.java` - Banka uygulaması örneği

## Çalıştırma

Java kaynakları bağımsız derleme ve çalıştırma için aşağıdaki komutu kullanabilirsiniz:

```bash
cd src/java
javac Application.java
java Application
```

veya `BankApplication.java` için:

```bash
cd src/java
javac BankApplication.java
java BankApplication
```

## Notlar

- Buradaki Java dosyaları proje örnekleri olarak tutulur.
- Java yapılandırması için ek `.properties` veya `pom.xml` dosyaları gerektiğinde eklenebilir.
