# JavaScript Dökümantasyonu

`src/javascript/` klasörü JavaScript kaynaklarını içerir.

## İçerik

- `frontend_demo.js` - frontend entegrasyon örneği
- `install_sdk.js` - SDK kurulum ve başlangıç scripti

## Kullanım

- `frontend_demo.js` dosyası, projeye ön yüz düzeyinde JS entegrasyonu için bir örnektir.
- `install_sdk.js` dosyası, SDK kurulumu ve başlangıç işlemlerini destekler.

## Nasıl Entegre Edilir

HTML sayfanıza eklemek için:

```html
<script src="../javascript/frontend_demo.js"></script>
```

veya Node.js içinde çalıştırmak için:

```bash
cd src/javascript
node install_sdk.js
```
