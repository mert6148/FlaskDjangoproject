# System Protection for python

Bu klasör Python uygulama ve yardımcı scriptleri içerir.

## Koruma Önerileri

- Python kodu gizli bilgileri içermemelidir.
- `api_demo.py`, `app.py`, `class.py` ve diğer scriptler güvenli yazılım prensiplerine uygun olmalıdır.
- Flask/Django yapılandırmalarında gizli anahtarlar `.env` ve ortam değişkenlerinde tutulmalıdır.

## Uygulama

- `python` klasöründeki tüm scriptler versiyon kontrolü altında tutulmalı.
- Üretim ortamına taşınmadan önce kod gözden geçirilmelidir.
