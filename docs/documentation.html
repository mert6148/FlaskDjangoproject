FlaskDjango Proje Dokümantasyonu

Bu proje, Flask ve Django arasında uygulama geçişlerini yönetmek için hazırlanmış örnek bir yapı sunar. Python ve JavaScript kombinasyonuyla, modüler bir şekilde sistemsel entegrasyon sağlanmıştır.

1. Proje Hakkında

İki ana modül içerir:

  • Flask: `app.py`, `auth/`, `dashboard/`, `sys/api/` gibi klasörler içinde API ve sunucu mantığı.
  • Django: `app/fastadmin/`, `model/`, `sys/` gibi yapılarda CRUD, yönetim ve modüler uygulama geçişi.

Proje, hem Flask hem de Django altyapılarını aynı repo içinde barındırarak dönüşümlü kullanım imkânı sunar.

2. Kurulum

2.1 Ortam Hazırlığı

python -m venv .venv
.venv\Scripts\activate
pip install -r auth/Flask-Django_python/requirements.txt

Gerekirse Django tarafı için ek bağımlılıkları da app/fastadmin/ ve model/ klasörlerinden kurun.

2.2 Çalıştırma

  • Flask: python app.py veya python auth/main.py
  • Django için yönetici: python manage.py runserver (varsa proje ayarlarında)

3. Proje Dosya Yapısı

  • index.html : Gösterge panelinin ana arayüzü.
  • app.py, main.py : Flask ana uygulama giriş noktaları.
  • auth/ : Kimlik doğrulama ve kullanıcı yönetimi.
  • model/ : Veri katmanı ve model soyutları.
  • sys/api/ : API uç noktaları ve sunucu altyapısı.
  • dashboard/ : Yönetim paneli arayüzü.

4. Özellikler

  • Flask/Django taşınabilir mimari.
  • Modüler proje dizini ve bileşenler.
  • Basit API ve arayüz entegrasyonu.
  • Çoklu servis geçiş testleri (test/ klasörü).

5. Dökümantasyon Notları

Bu sayfa, proje genelini ve hızlı başlangıç bilgilerini içerir. Her modül için daha derin dokümantasyon gerekiyorsa aşağıdaki aşamaları takip edin:

  1. Kaynak dizinleri gözden geçir: app/, auth/, sys/
  2. Veri modellerini analiz et: model/
  3. API uç noktalarını test et: sys/api/ ve dashboard/
  4. Test senaryolarını çalıştır: test/run_all_tests.py
