Kurulum Adımları

Ortam

python -m venv .venv
.venv\Scripts\activate
pip install -r auth/Flask-Django_python/requirements.txt
pip install django djangorestframework

Flask Başlat

SET MODE=flask
python auth/main.py

Django Başlat

SET MODE=django
python auth/main.py migrate
python auth/main.py runserver 0.0.0.0:8000

MySQL Bağlantısı

Ortam değişkeni olarak MySQL bağlantısını ayarlayın:

set MYSQL_DATABASE_URL=mysql+mysqlconnector://kullanici:sifre@127.0.0.1:3306/dbadi

Flask için genel DB check: GET /api/db-check
