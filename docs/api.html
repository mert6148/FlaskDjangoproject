API Uç Noktaları

Flask API

  • GET /api/kullanicilar
  • POST /api/kullanicilar - {"ad":"...","email":"..."}
  • GET /api/kullanicilar/<id>
  • PUT /api/kullanicilar/<id>
  • DELETE /api/kullanicilar/<id>
  • GET /api/urunler
  • POST /api/urunler - {"isim":"...","fiyat":...,"kullanici_id":...}

Django API

  • GET /api/kategoriler/
  • GET /api/urunler/
  • GET /api/urunler/stok_yok/
  • POST /api/urunler/<id>/stok_ekle/ - {"miktar":50}

Veritabanı Kontrolleri

  • GET /api/db-check (Flask MySQL/SQLite bağlantı kontrolü)
  • GET /api/status (genel sistem durumu)

Örnek JavaScript Kullanımı

async function fetchKullanicilar() {
  const res = await fetch('/api/kullanicilar');
  const data = await res.json();
  console.log(data);
}

async function kullaniciEkle(ad, email) {
  const res = await fetch('/api/kullanicilar', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({ad, email})
  });
  return res.json();
}
